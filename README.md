# WordPress Real Estate Website

This repository contains the source code for a **WordPress-based real estate website**.  
It is designed to showcase property listings, allow user inquiries, and support future features like workshops or online sales.

## Features

- Property listing pages with images and descriptions  
- Contact form for inquiries  
- Custom WordPress theme and plugins  
- Responsive design for desktop and mobile  
- Future-ready for e-commerce or booking features
